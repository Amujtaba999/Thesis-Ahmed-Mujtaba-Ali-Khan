# Automation Intensity and Investor Returns

This repository contains the code and supplementary outputs for my master's thesis:

**Automation Intensity and Investor Returns: A Cross-Sectional Portfolio Analysis**

## Contents

- `Rcode`: Full R script used for the empirical analysis
- `tables/`: Final tables used in the thesis
- `figures/`: Figures used in the thesis and appendix
- `Processed files`: Variable definitions, data sources and reproduction notes

## Software

The analysis was conducted using R version 4.5.1.


## Automation Proxy

Automation intensity is measured as:

log(1 + PP&E per employee)

This variable should be interpreted as a capital-based proxy for automation intensity rather than a direct measure of automation.
